JavaScript vs Java - Estudo Comparativo

O Desafio

O JavaScript foi criado em 1995 para que houvesse mais recursos de interatividade nas páginas HTML e do CSS. Muitos usuários confundem o JavaScript com a linguagem de programação JAVA. O desafio proposto é:

  1. Pesquisar a diferença entre o JavaScript e a linguagem JAVA.
  2. Quais empresas criaram o JavaScript e o JAVA, e quais são as que atualmente mantêm o desenvolvimento dessas linguagens?
  3. Criar um código HTML que tenha um botão que faça com que a página fique da cor AZUL, por meio de uma chamada de JavaScript.

1. Diferenças entre JavaScript e Java

Característica JavaScript Java
Tipo Linguagem de script interpretada Linguagem compilada
Tipagem Dinâmica (tipos determinados em tempo de execução) Estática (tipos definidos na declaração)
Execução Interpretada pelo navegador ou motor JavaScript Compilada para bytecode e executada na JVM
Uso principal Desenvolvimento web front-end, back-end (Node.js) Aplicações corporativas, sistemas embarcados, Android
Orientação a objetos Baseada em protótipos Baseada em classes (tradicional)
Dependência de plataforma Independente (roda em navegadores ou ambientes Node.js) "Write once, run anywhere" (com a JVM)
Memória Gerenciamento automático de memória Garbage collection automático

Embora o nome sugira uma relação próxima, JavaScript e Java são linguagens fundamentalmente diferentes, criadas para propósitos distintos. O JavaScript foi nomeado assim principalmente por questões de marketing, aproveitando a popularidade do Java na época de seu lançamento.

2. Empresas Criadoras e Mantenedoras

JavaScript

  • Criador: Brendan Eich na Netscape Communications Corporation em 1995
  • Nome original: Mocha, depois LiveScript, e finalmente JavaScript
  • Mantenedores atuais:
    • O padrão ECMAScript é mantido pelo Comitê Técnico 39 (TC39) da ECMA International
    • Participantes do TC39 incluem representantes de Mozilla, Google, Microsoft, Apple e outros
    • Implementações específicas são mantidas pelos desenvolvedores de navegadores (Google mantém o V8, Mozilla mantém o SpiderMonkey, etc.)

Java

  • Criador: James Gosling e equipe na Sun Microsystems em 1995
  • Mantenedores atuais:
    • Oracle Corporation (desde a aquisição da Sun Microsystems em 2010)
    • A comunidade OpenJDK (versão open-source com contribuições da Oracle, Red Hat, e outros)

3. Mudar a Cor da Página com JavaScript

Clique no botão abaixo para mudar a cor de fundo desta página para AZUL usando JavaScript:

Código HTML e JavaScript utilizado:

<!-- Botão com evento onclick que chama a função JavaScript -->
<button onclick="mudarCorParaAzul()">Mudar para Azul</button>

<script>
    // Função que muda a cor de fundo da página para azul
    function mudarCorParaAzul() {
        // Seleciona o elemento body e altera sua propriedade backgroundColor
        document.body.style.backgroundColor = "#0000FF";

        // Mensagem no console para confirmar a execução da função
        console.log("A cor de fundo foi alterada para azul!");
    }
</script>
