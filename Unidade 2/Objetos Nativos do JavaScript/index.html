Sistema de Catalogação de Livros Digitais

Este sistema utiliza expressões regulares para identificar e contar ocorrências de nomes de autores em textos, considerando regras específicas para nomes com partes que podem não começar com maiúsculas (como preposições).

Como funciona a expressão regular:

A expressão regular utilizada é capaz de identificar nomes de autores como "Machado de Assis" e "Pedro Álvares Cabral", considerando que algumas partes do nome podem começar com letras minúsculas (como preposições "de", "da", etc.).

/\b([A-Z][a-zÀ-ÿ]+)(\s+([a-zÀ-ÿ]+\s+|[A-Z][a-zÀ-ÿ]+\s+))*([A-Z][a-zÀ-ÿ]+)\b/g

Explicação da expressão:

  • \b - Define uma fronteira de palavra
  • ([A-Z][a-zÀ-ÿ]+) - Captura uma palavra que começa com letra maiúscula seguida de uma ou mais letras minúsculas, incluindo caracteres acentuados
  • (\s+([a-zÀ-ÿ]+\s+|[A-Z][a-zÀ-ÿ]+\s+))* - Captura zero ou mais ocorrências de espaços seguidos por:
    • Uma palavra com letras minúsculas (como "de", "da") OU
    • Uma palavra que começa com maiúscula
  • ([A-Z][a-zÀ-ÿ]+) - Termina com uma palavra que começa com letra maiúscula
  • \b - Define outra fronteira de palavra
  • g - Flag global que permite encontrar todas as ocorrências
